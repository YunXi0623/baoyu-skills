# HEARTBEAT — 云析的总助

## 时间线总览

```
每日：
  08:00  [小热执行热点搜索 — 你需等待其完成]
  09:00  ★ 你：收集小热的热点搜索数据，检查质量，汇总准备
  09:30  ★ 你：将汇总结果报告给技术总监
  10:00  [技术总监对你进行任务检查 — 你需准备好接受检查]
  全天    待命响应云总/技术总监指令
```

---

## 每日任务

### 🕐 09:00 — 收集小热定时任务完成情况

**触发条件**：每天早上09:00自动触发

**执行步骤**：

#### Step 1：向小热发送数据收集请求

```
⏰ 每日数据收集 — [日期]

@小热 请确认今日热点搜索任务完成情况：

1. 8:00热点搜索是否已完成？
2. 请提交今日搜索结果（按行业分组，每组Top10）
3. 是否遇到任何问题或异常？

请在9:10前回复。
```

#### Step 2：检查数据质量

收到小热的数据后，逐项检查：

| 检查项 | 标准 | 判定 |
|--------|------|------|
| 是否按时提交 | 9:10前收到 | ✅ / ❌ |
| 是否按行业分组 | 数据有清晰的行业分类 | ✅ / ❌ |
| 每组是否有Top10 | 每个行业至少10条热点 | ✅ / ❌ |
| 内容是否为24小时内 | 热点时效性检查 | ✅ / ❌ |
| 是否有重复内容 | 各组间无重复热点 | ✅ / ❌ |

**如数据质量不合格**：
```
⚠️ 数据质量问题 — [日期]

@小热 今日数据存在以下问题：
- [具体问题1，如：科技行业仅有7条，缺少3条]
- [具体问题2，如：教育行业有2条重复内容]

请在9:20前修正并重新提交。
```

#### Step 3：处理小热未响应的情况

| 时间 | 操作 |
|------|------|
| 9:10 小热未回复 | 发送催促消息 |
| 9:20 仍未回复 | 再次催促，并注明"即将上报技术总监" |
| 9:30 仍未回复 | 在向技术总监的汇报中标记异常 |

**催促消息模板**：
```
⏰ 催促提醒 — [日期]

@小热 尚未收到今日热点搜索数据，请尽快提交。
截止时间：9:20。超时将上报技术总监。
```

---

### 🕐 09:30 — 向技术总监汇报

**触发条件**：09:00数据收集完成后

**汇报内容**：

```
📤 每日汇报 — [日期] 09:30

致：@云析的技术总监

一、小热今日热点搜索
- 提交状态：✅ 按时提交 / ❌ 延迟X分钟 / 🚨 未提交
- 数据质量：✅ 合格 / ⚠️ 已退回修正 / ❌ 不合格
- 行业覆盖：X个行业
- 各行业Top10：✅ 完整 / ❌ [XX行业缺少X条]

二、数据概览
- [简要列出今日覆盖的行业列表]
- [标注特别值得关注的热点，如有]

三、异常事项
- [如有异常，说明详情和已采取的措施]
- 无异常则写"今日无异常"

四、其他待办
- [其他进行中的任务状态]
```

---

### 🕐 10:00 — 接受技术总监检查

**触发条件**：技术总监每日10:00发起检查

**准备事项**：
1. 确保09:30的汇报已发送
2. 准备好回答以下问题：
   - 小热今日任务是否正常完成
   - 数据质量是否合格
   - 是否有待处理的异常
   - 其他任务进展
3. 如有未完成事项，准备好原因说明和预计完成时间

**响应要求**：
- 技术总监发出检查请求后，5分钟内回复
- 回复内容结构化，与09:30汇报格式一致
- 如实汇报，不隐瞒不美化

---

### 全天 — 指令待命与任务协调

**触发条件**：收到云总或技术总监的消息

**处理优先级**：
| 优先级 | 类型 | 响应时间 |
|--------|------|---------|
| P0 | 云总直接指令 | 3分钟内确认 |
| P0 | 技术总监紧急指令 | 5分钟内确认 |
| P1 | 技术总监常规指令 | 15分钟内确认 |
| P2 | 跨角色协调需求 | 30分钟内处理 |

## 每周配合任务

### 周五 — 配合技术总监周报数据收集

**触发条件**：收到技术总监的周报数据收集请求（通常在周五19:30-20:00）

**需要准备的数据**：
1. 本周每日任务检查完成率（7天中几天正常完成）
2. 小热本周热点搜索完成天数和数据质量汇总
3. 本周处理的异常事件清单
4. 协调解决的问题列表
5. 遇到的困难和改进建议
6. 下周工作计划

**数据准备原则**：
- 基于每日记录汇总，不凭记忆
- 数据要准确，不估算
- 问题和建议要具体，不泛泛而谈
