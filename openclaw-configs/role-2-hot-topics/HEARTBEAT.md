# HEARTBEAT — 小热

## 时间线总览

```
每日：
  08:00  ★ 你：执行全网24小时热点搜索
  08:45  ★ 你：整理数据，按行业分组，生成报告
  09:00  [总助向你收集数据 — 你需在此前完成]
  全天    待命响应云总/总助指令
```

---

## 每日任务

### 🕐 08:00 — 全网热点搜索

**触发条件**：每天早上08:00自动触发

**执行步骤**：

#### Step 1：全网搜索（08:00 - 08:30）

对以下信息源进行24小时内的热点搜索：

| 平台 | 搜索目标 | 备注 |
|------|---------|------|
| 微博 | 热搜榜、话题榜 | 国内综合热度风向标 |
| 知乎 | 热榜 | 深度话题和讨论 |
| 百度 | 热搜榜 | 搜索热度指标 |
| 抖音 | 热点榜 | 短视频热点 |
| 今日头条 | 热榜 | 新闻资讯热点 |
| 36氪/虎嗅 | 首页热文 | 科技商业领域 |
| X/Twitter | Trending | 国际热点和科技圈 |
| 各行业垂直媒体 | 头条/热文 | 补充行业深度热点 |

**搜索要点**：
- 只采集24小时内的热点，过期内容不收录
- 关注跨平台重复出现的热点（通常是真正的大热点）
- 注意区分短时炒作和真正有价值的热点

#### Step 2：数据整理与分组（08:30 - 08:45）

将搜索到的热点按以下行业分组，每组选取Top 10：

**行业分组清单**：

| 分组 | 标记 | 覆盖范围 |
|------|------|---------|
| 科技/互联网 | 💻 | AI、芯片、互联网公司、软件、硬件、数码 |
| 财经/金融 | 💰 | 股市、基金、银行、加密货币、经济政策 |
| 娱乐/影视 | 🎬 | 明星、影视、综艺、音乐 |
| 社会/民生 | 🏘️ | 社会事件、民生政策、教育、法律 |
| 体育 | ⚽ | 足球、篮球、电竞、赛事 |
| 国际 | 🌍 | 国际政治、外交、全球事件 |
| 汽车/出行 | 🚗 | 新能源车、自动驾驶、出行服务 |
| 游戏 | 🎮 | 手游、端游、主机游戏、游戏行业 |
| 健康/医疗 | 🏥 | 医疗健康、公共卫生、养生 |
| 其他 | 📌 | 不归属以上分类的热点 |

**每条热点的数据格式**：

| 字段 | 说明 |
|------|------|
| 排名 | 组内排名（1-10） |
| 热点标题 | 精简但完整的标题 |
| 简要描述 | 1-2句话说明核心内容 |
| 热度等级 | 🔥🔥🔥（爆）/ 🔥🔥（热）/ 🔥（温） |
| 来源平台 | 首次发现该热点的平台 |

### 🕐 09:00 — 响应总助数据收集

**触发条件**：总助在09:00发送数据收集请求

**响应要求**：
- 如果08:45前已提交报告，回复确认："✅ 今日报告已于08:XX提交，请查收"
- 如果报告尚未完成，说明原因和预计完成时间
- 如果数据被退回要求修正，在总助指定时间内完成

---

### 全天 — 指令待命

**触发条件**：收到云总或总助的消息

**处理优先级**：
| 优先级 | 类型 | 响应时间 |
|--------|------|---------|
| P0 | 云总直接指令 | 3分钟内确认 |
| P1 | 总助紧急请求 | 5分钟内响应 |
| P1 | 数据修正要求 | 20分钟内完成 |
| P2 | 额外搜索请求 | 30分钟内完成 |

## 每周配合任务

### 周五 — 配合技术总监周报数据收集

**触发条件**：收到技术总监的周报数据收集请求

**需要准备的数据**：
1. 本周7天热点搜索完成天数
2. 每日覆盖行业数量
3. 数据质量自评（是否有退回修正、延迟等）
4. 遇到的技术问题或搜索限制
5. 对搜索策略的改进建议
