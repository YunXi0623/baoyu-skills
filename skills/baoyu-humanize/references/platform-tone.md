# Platform Tone Guide（各平台自然语感参考）

## Xiaohongshu（小红书）

### Tone DNA
口语化、亲切、有情绪但不浮夸、像闺蜜/朋友聊天

### What natural XHS posts sound like

```
昨天试了这个工具，一开始还觉得不靠谱，结果出来的效果把我惊到了

不是广告哈，纯分享，好不好用你们自己试

用了一周了，说说真实感受：没有网上说的那么神，但确实省了不少事
```

### Characteristics

| Feature | Details |
|---------|---------|
| Sentence length | Short, 10-20 characters per sentence |
| Punctuation | Casual, can skip periods, use ... for trailing thoughts |
| Paragraph length | 1-3 sentences, lots of line breaks |
| Emotion markers | 哈哈、绝了、真的会谢、救命、姐妹们 |
| Hedging | "个人感受哈"/"仅供参考"/"因人而异" |
| First person | Heavy use of 我、我的 |
| Colloquial | 搞、整、折腾、薅羊毛 |

### Avoid on XHS
- Long paragraphs (nobody reads them)
- Formal/academic tone
- English-heavy content (unless audience is bilingual)
- Excessive hashtags in body text

## WeChat Official Account（微信公众号）

### Tone DNA
有温度的书面语、叙事感、专业但不学术、像一个见多识广的朋友在跟你聊

### What natural WeChat articles sound like

```
上周在公司��到一个挺有意思的事。我们组的一个实习生，用ChatGPT帮自己写了份周报，
结果被领导夸了。这事本身不稀奇，稀奇的是——他用的那个prompt，是从一个付费社群里学来的。

这让我开始想一个问题：AI工具本身在变便宜，但"会用AI"这件事，反而在变值钱。
```

### Characteristics

| Feature | Details |
|---------|---------|
| Sentence length | Medium, 15-40 characters |
| Paragraph length | 2-4 sentences, then line break |
| Narrative style | Scene → observation → insight (not conclusion) |
| Personal markers | "我发现"/"我的感受是"/"说实话" |
| Data use | Specific numbers add credibility: "38美元"/"2小时" |
| Headings | H2 for sections, sparingly |
| Bold | Key phrases only, not full sentences |

### Avoid on WeChat
- Bullet-point-only articles (not WeChat culture)
- Starting with "今天给大家分享"
- Ending with "希望对你有所帮助"
- Excessive emojis
- Front-loading the conclusion

## X / Twitter

### Tone DNA
简短直接、有态度、不装、像发短信给朋友

### What natural X posts sound like

```
折腾了一晚上终于跑通了。坑不少，但结果还行。

用了3个月 Claude，说句大实话：写代码确实比 GPT 强，但其他方面差距没网上说的那么大。

副业第三个月，工具站收到第一笔付费。没有特别激动，更多是确认了这条路能走。
```

### Characteristics

| Feature | Details |
|---------|---------|
| Length | 60-220 characters for Chinese |
| Tone | Casual, direct, opinionated |
| Structure | No structure, just talk |
| Hashtags | 0-2, only if natural |
| Emoji | 0-1, sparingly |
| Thread style | Each tweet standalone, but builds on previous |
| Ending | Open-ended or just stops, no CTA |

### Avoid on X
- Thread where every tweet starts with a number ("1/")
- Promotional language ("关注我获取更多")
- Excessive emojis (🔥🚀💡 = instant AI detection)
- Perfect grammar and structure (too polished = suspicious)
- Generic opinions ("AI will change everything")

## Cross-Platform Adaptation

When humanizing the same content for different platforms:

| Aspect | XHS | WeChat | X |
|--------|-----|--------|---|
| Same fact | "这个工具居然免费！救命！" | "有意思的是，这个工具目前还是免费的。" | "这工具免费的，先用着。" |
| Uncertainty | "不确定哈，你们试试看" | "我也还在观察，目前来看..." | "还在看，先不下结论。" |
| Recommendation | "姐妹们冲就完事了" | "个人觉得值得一试，尤其是..." | "推荐试试，适合 XX 场景。" |
| Conclusion | "好啦就分享这么多~" | (直接停，不用结尾) | (直接停) |
