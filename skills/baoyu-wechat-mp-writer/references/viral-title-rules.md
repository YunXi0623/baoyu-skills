# 爆款标题生成规则

## 核心公式

**爆款标题 = 直击人性 + 吸睛词汇 + 顺口句式**

## 生成要求

1. 每篇文章输出 **5 个候选标题**，标注所用手法类型
2. 标题字数 **不超过 50 字**
3. 采用以下手法煽动用户情绪：提问、金钱、数字、暴力、捷径、异常、悬念、损失心理、反常识
4. 多使用情感强烈的词汇或流行词汇
5. 标题需与文章内容匹配，不能纯标题党（内容要能接住标题）

## 8 种标题类型

### 1. 悬念/猎奇型
只把话说一半，引发好奇心。
- 《很严重，大家开始存钱吧》
- 《博士入职比亚迪，自爆真实工作感受》
- 《家庭变故，嫂子供我上学10年，如今我年薪100万想报答嫂子，才知她骗了我15年》

### 2. 数字型
数字锁定注意力，更直观具体。
- 《这个技巧，99%的程序员都不知道》
- 《月薪3千与月薪3万的文案，差别究竟在哪里？》
- 《这篇文章只改了下标题，阅读量就从1万+涨到10万+》

### 3. 矛盾冲突型
制造反差和冲突，调动好奇心。
- 《打工人的8天假，比上班还累》
- 《他去世了，年入200亿，不给子女留1分钱》
- 《高考故意考0分的学霸，早已年入百万》

### 4. 问题型
干货类文章或解答具体问题。
- 《为什么杀人容易分尸难》
- 《中年人的简历，狗都嫌》
- 《网约车里为什么总是臭臭的》

### 5. 符号型
感叹号增强感官刺激。
- 《腾讯下狠招：这类的微信号全部都要封号！》
- 《重磅！万一打狗队上门来我家抓狗怎么办？攻略奉上不谢》

### 6. 抱大腿型
借势名人/热点事件自带流量。
- 《雷军推荐的程序员必看的十本好书》
- 《比起汪峰离婚，这位中国护士的行为才应该上头条》

### 7. 代入感型
让读者感觉"说的就是我"。身份标签（年龄、职业、星座）或情感共鸣。
- 《人过40，不管在谁面前，都不能亮出这3张底牌，真的很蠢》
- 《生活不止诗和远方，还有傻逼甲方》

### 8. 人性弱点型
利用傲慢、嫉妒、暴怒、懒惰、贪婪、色欲。
- 《为何学习比你差的同学都当老板了，你却还在打工》
- 《我为什么放弃百万年薪》
- 《免费！限前100名，手慢无》

## 输出格式

```
### 候选标题

| # | 标题 | 手法 |
|---|------|------|
| 1 | 《xxx》 | 数字+悬念 |
| 2 | 《xxx》 | 矛盾冲突 |
| 3 | 《xxx》 | 人性弱点+代入感 |
| 4 | 《xxx》 | 抱大腿+问题 |
| 5 | 《xxx》 | 符号+猎奇 |

**推荐**：#N，理由：xxx
```

## 注意事项

- 技术类文章也要有情绪张力，纯干货标题没人点
- 避免与合规审核冲突（不用绝对化用语如"最强""第一"）
- 标题里的数字尽量具体（"50条"比"很多条"好，"3.5小时"比"半天"好）
- 可以适度夸张但不能虚假，文章内容要能接住标题的承诺
