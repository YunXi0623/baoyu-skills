# Common Rules（通用违规规则）

Rules that apply across all platforms. Platform-specific rules are in `platforms/` subdirectory.

## 1. Advertising Law Absolute Claims（广告法绝对化用语）

Severity: **High** — may trigger platform review or removal

### Superlative / Ranking Claims

| Prohibited | Suggested Replacement |
|-----------|----------------------|
| 最好 / 最佳 / 最优 | 很好 / 非常好 / 相当不错 |
| 最大 / 最小 / 最强 | 很大 / 较小 / 很强 |
| 第一 / NO.1 / Top1 | 领先 / 头部 / 前列 |
| 唯一 / 独一无二 | 少见 / 很特别 / 有特色 |
| 顶级 / 顶尖 / 极致 | 高端 / 出色 / 优秀 |
| 国家级 / 世界级 | 知名 / 专业级 |
| 绝对 / 绝无仅有 | 非常少见 / 很难得 |
| 万能 / 全能 | 多功能 / 用途广 |
| 首选 / 首创 / 首发 | 推荐 / 早期推出 |
| 史上最 / 全网最 | 非常 / 很 |

### Absolute Guarantee Claims

| Prohibited | Suggested Replacement |
|-----------|----------------------|
| 100% / 百分百 | 大概率 / 很大可能 |
| 保证 / 承诺 / 必定 | 预期 / 大概率 / 通常 |
| 永久 / 终身 / 永远 | 长期 / 持续 |
| 零风险 / 无风险 | 低风险 / 风险较小 |
| 立竿见影 / 立即见效 | 较快见效 / 短期内 |

## 2. False/Exaggerated Claims（虚假夸大宣传）

Severity: **High**

| Prohibited Pattern | Why | Suggested Fix |
|-------------------|-----|---------------|
| 效率提升 N 倍 (N>10) | Exaggeration | 降低倍数或改为"大幅提升" |
| 颠覆 / 革命性 / 划时代 | Overhype | 改为"创新"/"很有意思" |
| 秒杀一切 / 碾压 / 吊打 | Aggressive comparison | 改为"表现不错"/"有优势" |
| 人人都在用 / 全民 | Unverifiable | 改为"很多人在用"/"不少人试过" |
| 月入 N 万（无实证） | Income claims without proof | 改为"有收入潜力"/"有变现空间" |

## 3. Medical / Health Claims（医疗健康类）

Severity: **High** — regulated content

| Prohibited Pattern | Why |
|-------------------|-----|
| 治疗 / 治愈 / 根治 | Medical claims require qualification |
| 药到病除 / 包治百病 | False medical promise |
| 减肥 N 斤 / 快速瘦身 | Health claims without evidence |
| 排毒 / 养颜 / 抗衰老 | Pseudo-medical terms |

Replacement strategy: Remove medical claims entirely or add disclaimer "非医疗建议".

## 4. Financial / Investment Claims（金融投资类）

Severity: **High** — regulated content

| Prohibited Pattern | Why |
|-------------------|-----|
| 保本 / 稳赚 / 零亏损 | Financial guarantee |
| 收益率 N% | Specific return promise |
| 暴富 / 躺赚 / 被动收入 | Misleading income claims |
| 日赚 / 月赚 + 具体金额 | Unverifiable income claims |

## 5. Political Sensitivity（��治敏感）

Severity: **High** — may result in account ban

General rule: Avoid discussing political figures, policies, national events in a non-neutral way. This skill flags obvious patterns but cannot replace human judgment on political sensitivity.

## 6. Urgency / Scarcity Manipulation（紧迫感操纵）

Severity: **Medium**

| Prohibited Pattern | Suggested Replacement |
|-------------------|----------------------|
| 最后 N 个名额 | 名额有限 |
| 限时免费 / 马上涨价 | 目前可用 / 当前价格 |
| 不买后悔 / 错过就没了 | 感兴趣可以了解一下 |
| 仅剩 / 即将售罄 | 数量有限 |
