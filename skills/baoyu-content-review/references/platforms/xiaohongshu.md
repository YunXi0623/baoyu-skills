# Xiaohongshu Rules（小红书违规规则）

## 1. Traffic Diversion / Contact Info（引流违规）

Severity: **High** — triggers throttle or shadow ban

| Prohibited | Suggested Replacement |
|-----------|----------------------|
| 微信 / wx / WX / VX / vx | 🍑（谐音梗）or just remove |
| 微信号 / 加我微信 | "主页有联系方式" or remove |
| QQ / qq号 | Remove |
| 私信我 / 私我 / 滴滴我 | "评论区聊" |
| 评论区找我 / 扣1 | "有问题欢迎讨论" |
| 淘宝 / 天猫 / 京东 / 拼多多 | "电商平台" or remove |
| 链接在 bio / 主页链接 | Remove |
| 关注公众号 | Remove or "有更多内容在别的平台" |

### Variants to Watch

These are common evasion patterns that XHS algorithm also detects:
- V信 / 威信 / 薇信 / 加V
- 绿色软件（refers to WeChat）
- 某宝 / 某东 / 某多多
- 私 / si / S我

## 2. Engagement Bait（诱导互动）

Severity: **Medium** — may reduce distribution

| Prohibited | Why |
|-----------|-----|
| 求关注 / 关注我 / 记得关注 | Explicit follow begging |
| 必转 / 快转发 / 分享给朋友 | Share manipulation |
| 点赞收藏 / 三连 / 一键三连 | Engagement manipulation |
| 不看后悔 / 建议收藏 | Low-quality urgency |

Suggested: Let content quality drive engagement naturally. If you must, use subtle hints like "觉得有用的话可以存一下".

## 3. Sensitive Product Categories（敏感品类）

Severity: **Medium to High**

| Category | Flagged Terms | Notes |
|----------|--------------|-------|
| Weight loss | 减肥、瘦身、掉秤、暴瘦 | Must add "个人体验，非医疗建议" |
| Money/income | 赚钱、暴富、变现、月入N万 | Needs realistic framing |
| Gambling | 彩票、赌、下注、中奖 | Strictly prohibited |
| Tobacco/alcohol | 烟、酒（in promotional context） | Age-restricted |
| Adult content | Any sexual or suggestive content | Strictly prohibited |

## 4. Title / Cover Bait（标题党）

Severity: **Medium** — reduces distribution quality

| Pattern | Example | Fix |
|---------|---------|-----|
| Excessive punctuation | ！！！/ ？？？ | Max 1 exclamation/question mark |
| ALL CAPS clickbait | 震惊！太疯狂了！ | Tone down, be specific |
| Misleading thumbnail | Title doesn't match content | Align title with actual content |
| Emoji spam | 🔥🔥🔥💰💰💰 | Max 2-3 emojis, naturally placed |

## 5. Competitor Mentions（竞品提及）

Severity: **Low to Medium** — may reduce distribution

Direct comparison posts mentioning competitor brand names (e.g., "比XX好用") may get less traffic. Suggested approaches:
- Use generic terms: "同类产品"/"其他工具"
- Focus on your own experience rather than comparison
- If comparing, be factual and neutral

## 6. XHS Content Format Rules

Not "prohibited" but important for distribution:
- Image-first platform: text-only posts get minimal reach
- 3-6 images optimal
- First image = cover, must be eye-catching
- Text should be under 300 characters for best engagement
- Hashtags: 3-5, mix of popular and niche
